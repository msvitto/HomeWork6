виды селекторов
вид пример значение
универсальный селектор *{
color:red;
}		 применяеться к любому элементу
сектор по id #heading{
color:red;
}		 применяеться к индивидуальному элементу у которого есть этот id
селектор по class .title{
color:red;
}		 выбирает элемент, у которого есть атрибут class с указаним значением
селектор по названию тега h1{
color:red;
}		 примениться ко всем указаным элементам на странице (пример h1)
несколько селекторов h1, a, p{
color:red;
}		 указывает свойства стилей для всех выбраных селекторов
селектор по атрибуту [href="#"]{
color:red;
}		 применится ко всем элементам с таким атрибутом
селекторы потомков/контексткые селекторы .wraping p{
color:red;
}		 примениться ко всем дочерним элементам всех уровней
селекторы дочерние .wraping>p{
color:red;
}		 применяеться к первому уровню дочерних элементов
селекторы сестринские/соседние.смежные h1+p{
color:red;
}		 примениться к последнему указанному в селекторе элементу
селекторы сестринские/соседние.смежные h1~p{
color:red;
}		 примениться ко всем следующим элементам, не только к первому элементу
селекторы псевдоклассов a:hover{
color:red;
}		 они описывают состояние элементов при совершение над ними действия (клик по елементу)
селекторы псевдоэлементов p::first-letter{
color:red;
}		 они для добавления элементов на страницу без отображения их в HTML-разметке
